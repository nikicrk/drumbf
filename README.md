untuk menghapus script yg lama silahkan ketik :
````php
$ rm -rf mbfb
````
### Install Script
````php
$ pkg update && pkg upgrade
$ pkg install python 
$ pip install mechanize
$ pkg install git
$ git clone https://github.com/Mark-Zuck/rombf
$ cd rombf
$ pip install -r requirements.txt
$ python rom.py
````
